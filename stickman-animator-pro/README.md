# StickMan 3D Animator Pro

Una aplicación avanzada de animación 3D para stickman, construida con tecnologías modernas.

## 🚀 Tecnologías Utilizadas

- **React 19** - Biblioteca de UI moderna
- **TypeScript** - Tipado estático para mayor seguridad
- **Vite** - Build tool ultrarrápido
- **Three.js** - Renderizado 3D en el navegador
- **CSS Moderno** - Estilos con gradientes y efectos avanzados

## ✨ Características Principales

### Personalización del StickMan
- 🎨 Selector de color personalizado
- 📐 Ajuste de proporciones (cabeza, cuerpo, brazos, piernas)
- 👁️ Control de cámara con zoom

### Animaciones
- 🎭 8 animaciones predefinidas:
  - Idle (reposo)
  - Walk (caminar)
  - Run (correr)
  - Wave (saludar)
  - Jump (saltar)
  - Dance (bailar)
  - Punch (golpear)
  - Kick (patear)
- ⚙️ Control de velocidad de animación
- 🎛️ Control manual de articulaciones

### Exportación
- 📹 Grabación de video en tiempo real
- Formatos WebM y MP4
- Múltiples calidades (480p, 720p, 1080p)
- FPS configurables (24, 30, 60)
- 📸 Capturas de pantalla en PNG

## 🛠️ Instalación

```bash
cd stickman-animator-pro
npm install
```

## 🏃 Ejecución

### Modo Desarrollo
```bash
npm run dev
```

### Construcción para Producción
```bash
npm run build
```

### Vista Previa de Producción
```bash
npm run preview
```
